# N5 Technicall Test

Serie Two Project

## Installation

Use the package manager [yarn](https://yarnpkg.com/) or [npm](https://www.npmjs.com/) to install the project.

```bash
npm install
```
or

```bash
yarn install
```
## Usage

Run the command start to run the project on port [9000](http:localhost:9000)

```bash
npm run start
```
or

```bash
yarn start
```
